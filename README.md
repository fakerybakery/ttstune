# TTSTune

Configuration-driven framework for fine-tuning TTS models.

## Roadmap

- Chatterbox
- F5-TTS
- StyleTTS 2

## Development

```bash
pip install uv
uv venv
source .venv/bin/activate
uv pip install -r requirements.txt
pre-commit install
```
